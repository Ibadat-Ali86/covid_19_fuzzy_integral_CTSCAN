# Historical Research Records

This directory stores **author-supplied archival results**, not outputs generated by the code. The training and evaluation pipeline never copies these values into a run folder.

For an exact reproduction claim, archive the following together:

- frozen `split_manifest.csv` with SHA-256 hashes;
- teacher and student checkpoints;
- `resolved_config.yaml`;
- topology cache signature and train-only standardizer;
- package/environment capture;
- raw validation and test prediction CSV files.

`TopoFM-Slice-v1` is deliberately named as a slice-level experiment. It must not be described as the complete TopoFM-DG system because the archived run did not activate VREx, domain-adversarial learning, soft masks, multi-domain training, patient-level 2.5D aggregation, external validation, or ensembling.
